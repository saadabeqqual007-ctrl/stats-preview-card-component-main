# Stats Preview Card Component

A simple and responsive stats preview card component built with HTML and CSS.

## Screenshot

stats-preview-card-component/screenshot

## Features

* Responsive design
* Semantic HTML5
* CSS Grid and Flexbox layout
* Mobile-first approach
* Overlay effect using `mix-blend-mode`
* Clean and organized CSS structure
* Google Fonts integration

## What I Learned

Through this project, I practiced:

* Structuring a responsive card component with semantic HTML.
* Building layouts using both CSS Grid and Flexbox.
* Applying a mobile-first workflow with media queries.
* Creating a colored image overlay using `mix-blend-mode` and opacity.
* Organizing CSS into reusable and easy-to-read sections.
* Working with custom fonts from Google Fonts.
* Improving spacing, alignment, and typography for different screen sizes.
